#  Skin Advisor Telegram Bot

![Bot Logo](https://i.imgur.com/example.png) **Skin Advisor** — это ваш персональный косметолог-консультант в Telegram. Бот проводит детальную диагностику кожи на основе анкеты и формирует персональные рекомендации по уходу и макияжу со ссылками на покупку.

## 🚀 Основные возможности

* **🧪 Детальная анкета:** Пошаговый опрос для определения типа, состояний и особенностей вашей кожи.
* **📊 Точная диагностика:** Алгоритм определяет тип кожи, подтон, колорит и ключевые состояния (обезвоженность, чувствительность, акне и т.д.).
* **🛍️ Персональные рекомендации:** Бот подбирает утренний, вечерний и еженедельный уход, а также декоративную косметику.
* **🛒 Ссылки на покупку:** Для каждого продукта предоставляются ссылки на популярные маркетплейсы (`Ozon`, `Wildberries`, `Золотое Яблоко`) в трех ценовых категориях.
* **📜 Правила совместимости:** Бот предупреждает о правилах введения активных ингредиентов (ретиноиды, кислоты) и их сочетаемости.
* **📄 Экспорт результатов:** Возможность скачать полный отчет о диагностике и рекомендациях в формате PDF.
* **🔐 Безопасность и приватность:** Бот не собирает лишних данных и работает в соответствии с политикой конфиденциальности.

## ⚙️ Как пользоваться

Просто начните диалог с ботом в Telegram: [Ссылка на вашего бота]

### Основные команды

* `/start` — Начать работу с ботом.
* `/reset` — Сбросить анкету и начать диагностику заново.
* `/results` — Показать последние результаты диагностики.
* `/export` — Скачать результаты в PDF.
* `/help` — Получить помощь.
* `/privacy` — Ознакомиться с политикой конфиденциальности.

## 🛠️ Для владельца бота (Checklist)

Перед запуском бота в публичный доступ, пожалуйста, выполните следующие шаги:

* [ ] **Получить токен:** Создайте нового бота в `@BotFather` и вставьте полученный токен в файл `.env`.
* [ ] **Настроить администраторов:** Укажите ваш Telegram User ID в переменной `ADMIN_IDS` в `.env` для доступа к админ-панели.
* [ ] **Выбрать режим запуска:** Установите `RUN_MODE` в `.env` (`polling` для тестов, `webhook` для продакшена).
* [ ] **Обновить каталог продуктов:** Внимательно проверьте и обновите файл `data/products.json`. Убедитесь, что ссылки и артикулы актуальны.
* [ ] **Проверить конфигурацию:** Откройте `config/app.yaml` и настройте параметры под ваш проект (бренды, маркетплейсы, тексты).
* [ ] **Проверить дисклеймер:** Убедитесь,что текст дисклеймера в `config/app.yaml` соответствует вашим требованиям.
* [ ] **Протестировать все сценарии:** Пройдите анкету для разных типов кожи (сухая, жирная, с акне и т.д.), чтобы убедиться в корректности рекомендаций.
